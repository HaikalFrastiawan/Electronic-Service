# Electronic Service Booking System

Electronic Service adalah platform manajemen layanan perbaikan barang elektronik yang dibangun dengan **Golang** (Back-end) dan **React** (Front-end). Project ini dirancang dengan arsitektur yang bersih, performa tinggi, dan skalabilitas untuk UMKM jasa servis.

## 🚀 Tech Stack
- **Back-end:** Golang (Gin Gonic Framework)
- **Database:** PostgreSQL
- **ORM:** GORM
- **Front-end:** React.js (Planned)

## 📁 Project Structure
Struktur folder mengikuti standar *Clean Architecture* sederhana:
- `/config`: Konfigurasi database & environment.
- `/models`: Definisi skema database (struct).
- `/controllers`: Logika bisnis dan handler API.
- `/routes`: Manajemen endpoint URL.
